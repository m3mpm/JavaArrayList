# Java ArrayList Implementation

## Описание

Этот проект представляет собой простую реализацию интерфейса `ArrayList` на языке Java, который управляет коллекцией объектов типа `Car`. Реализация включает основные операции, такие как добавление, удаление, получение элементов и управление размером списка.

## Функциональность

### Интерфейс `CarListInterface`

Интерфейс определяет следующие методы:

- `Car get(int index)`: Получить элемент по индексу.
- `void add(Car car)`: Добавить элемент в конец списка.
- `void add(Car car, int index)`: Добавить элемент по указанному индексу.
- `void addFirst(Car car)`: Добавить элемент в начало списка.
- `void addLast(Car car)`: Добавить элемент в конец списка.
- `boolean remove(Car car)`: Удалить элемент по значению.
- `boolean remove(int index)`: Удалить элемент по индексу.
- `int size()`: Получить текущее количество элементов в списке.
- `void clear()`: Очистить список.

### Класс `CarArrayList`

Класс `CarArrayList` реализует интерфейс `CarListInterface` и использует массив для хранения объектов `Car`. Он автоматически увеличивает размер массива при необходимости.

## Тестирование

Проект включает набор тестов для проверки функциональности класса `CarArrayList`. Тесты написаны с использованием JUnit и проверяют:

- Получение элементов по корректным и некорректным индексам.
- Добавление элементов в различные позиции.
- Удаление элементов по значению и индексу.
- Очистку списка.

## Использование

Вы можете использовать класс `CarArrayList` для создания и управления списком объектов `Car`. Пример использования:

```Java
CarListInterface carList = new CarArrayList();
carList.add(new Car("Toyota", 2020));
carList.add(new Car("Honda", 2021));
Car myCar = carList.get(0);
System.out.println("Моя машина: " + myCar.getBrand());
```
